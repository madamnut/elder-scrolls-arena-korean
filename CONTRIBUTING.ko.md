# 기여 안내

## 시작하기 전에

- 정식 Steam판 Arena 1.07 CD 설치본을 사용한다.
- Bethesda 원본 파일과 추출 이미지를 이슈·PR·커밋에 첨부하지 않는다.
- 공개된 번역문과 직접 작성한 코드만 제출한다.
- 화면 문제는 가능한 한 전체 DOSBox 스크린샷과 재현 순서를 제공한다.

## 작업 종류

### 번역

1. [번역 규칙](docs/TRANSLATION.ko.md)을 읽는다.
2. 자리표시자와 제어 문자를 보존한다.
3. AKC 빌드를 실행한다.
4. 320×200 실기 화면에서 줄바꿈을 확인한다.

### 코드

1. 입력 원본 해시 검증을 우회하지 않는다.
2. 원본을 직접 덮어쓰는 기능을 추가하지 않는다.
3. 출력은 임시 파일에서 검증한 뒤 이동한다.
4. 재현 명령과 검증 결과를 PR에 기록한다.

### 이미지

원본 그림이 포함된 파일은 공개 PR에 올리지 않는다. 이미지 작업은 비공개 작업 공간에서 진행하고, 공개 저장소에는 변환 도구·좌표·해시·델타 생성 절차만 제출한다.

## 이슈 양식

```text
패치 버전:
게임 배포판과 언어:
원본 ACD.EXE SHA-256:
화면 순서:
재현 절차:
예상 결과:
실제 결과:
DOSBox 완전 재시작 여부:
스크린샷:
```

## PR 체크리스트

- [ ] 원본 게임 파일이 포함되지 않았다.
- [ ] 생성물과 분석 결과가 포함되지 않았다.
- [ ] 번역 자리표시자가 보존됐다.
- [ ] 관련 빌더가 성공했다.
- [ ] 실기 또는 포맷 단위 검증을 수행했다.
- [ ] 문서와 변경 기록을 갱신했다.

## 커밋 메시지

```text
feat: 기능 또는 번역 추가
fix: 표시·충돌·데이터 오류 수정
docs: 문서 변경
test: 검증 추가
refactor: 동작을 바꾸지 않는 구조 개선
build: 빌드·배포 변경
```
