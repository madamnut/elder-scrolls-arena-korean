# 변경 기록

이 프로젝트는 공개 릴리스 전 개발 단계다. 날짜는 한국 표준시를 기준으로 기록한다.

## 미출시

### 추가

- AKC 2바이트 완성형 한글 인코딩
- DOS TSR 한글 렌더러와 EMS 글꼴 적재
- 원본 EXE 폭 계산·그리기 루틴 `INT 60h` 연결
- BSA, IMG, FLC, DFA, 폰트 분석·변환 도구
- 타이틀과 캐릭터 생성 이전 이미지 화면 한글화
- 캐릭터 생성 방식 선택과 질문 안내문
- 직업 판정 질문 40개 및 선택지 120개 번역
- 별도 `QUEST_KR.TXT` 로딩
- 설치·복구·배포·개발·번역·시험 문서

### 변경

- 게임 제목은 한국어로 표시하고 원본 저작권 문구는 유지
- `SCROLL03.IMG`는 실제 CD판 느슨한 파일을 기준으로 처리
- 큰 글꼴의 한글 표시 높이를 최대 8픽셀로 제한

### 알려진 문제

- 작은 `FONT_B` 질문 화면에서 한글이 조밀하게 보임
- 직업 추천 결과와 직접 선택 목록은 아직 영어
- 캐릭터 생성 이후 대부분의 본편 텍스트는 미번역
- 한글 이름 입력은 지원하지 않음
- 배포용 단일 CMD 패처는 설계 단계
